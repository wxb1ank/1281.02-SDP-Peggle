# Peggle

## Building

### Prerequisites

- GNU Make
- GCC (both `g++` and `gcc` must be available)
- Git
- Doxygen (if generating documentation)

### Procedure

Run `make` or `make all` in this directory. The product will be written to either `./game` or `./game.exe` depending on the host platform.
